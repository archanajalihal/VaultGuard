# VaultGuard Pro 🛡️

**VaultGuard Pro** is an enterprise-grade, offline-first personal credential manager designed for maximum privacy and security. It runs entirely in your browser using local encryption keys—your data never leaves your device.

[**Access VaultGuard Pro Here**](https://archanajalihal.github.io/VaultGuard/)

## 🛡️ Key Security Features
- **Zero-Knowledge Architecture**: No cloud, no databases, no tracking. Total privacy.
- **PBKDF2 Key Derivation**: Uses 10,000 iterations to derive highly secure encryption keys from your master password.
- **AES-256 Encryption**: Industry-standard encryption for all stored credentials, API keys, and notes.
- **Security Audit Logs**: Track every login attempt, entry modification, and data export.
- **Session Auto-Lock**: Automatically seals the vault after 60 seconds of inactivity to prevent unauthorized access.
- **Emergency Recovery**: Generate a physical 256-bit recovery string for offline backup.

## 🎨 Professional Design
- **High-End UI**: Minimalist technical aesthetic with "Inter" and "JetBrains Mono" typography.
- **Stealth Mode**: Automated blurring of sensitive data to prevent shoulder-surfing.
- **Mobile Responsive**: Secure access across desktop and mobile browsers.

## 🛠️ Usage
1. Open the application.
2. **Initialize**: Set a strong Master Password (min 12 characters).
3. **Manage**: Add, edit, and organize your credentials with precision.
4. **Backup**: Export your encrypted "Cold Backup" JSON file regularily.

## 🔒 Technologies Used
- **Logic**: Vanilla JavaScript (ES6+)
- **Cryptography**: CryptoJS (PBKDF2, AES-256)
- **Styling**: Modern CSS3 (Custom Properties, Flexbox, Grid)
- **Icons**: Lucide Icons
